Easy Math for Kids
Web Application

I helped develop and launch a web application that offers online abacus lessons to children in school. Built using React, NextJS, Spring Security, and an SQL database hosted on a Google Cloud server, the app features interactive videos, lessons, and tutorials designed to engage young learners and help them develop their abacus skills. As part of my contribution to the project, I developed the user interface for teachers, which allows them to unlock lesson modules for their students.
